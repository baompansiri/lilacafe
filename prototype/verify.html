💡 วิธีใช้:
  1. ทุกการ์ดมี 3 ปุ่ม: ✓ ตรง · ✗ ผิด · ? ไม่แน่ใจ — กดเลือก state
  2. ✗ ผิด → กล่อง "ชื่อจริง" เปิด พิมพ์ชื่อถูกต้องลงไป
  3. 📷 Upload → เลือกรูปจากเครื่อง · resize อัตโนมัติเป็น 1200px JPEG · เก็บใน browser
  4. เมนูที่ยังไม่มีรูป (no-photo) → กด upload ใส่ได้เลย
  5. + เพิ่มเมนูใหม่ ที่ส่วนล่างของแต่ละหมวดถ้ามีเมนูไม่อยู่ใน list
  6. เสร็จแล้วกด 💾 Download ZIP (มีรูป + JSON) ส่งให้ Claude หรือ 📋 Copy Markdown paste ในแชท
